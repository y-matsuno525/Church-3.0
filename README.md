# Church-3.0 (旧verは[こちら](https://github.com/y-matsuno525/Church-2.0))
## 概要
教会に属さない人のためのキリスト教系コミュニティサイト。正典が不確定であること、AIによる教義生成が特徴。
## 機能
- 聖句単位での注釈の作成、共有
- 聖句単位でのQ&A
- （未完成）複数バージョンの聖書や外典、偽書の利用
- （未完成*）論文投稿機能
- （未完成）聖書内検索
- （未完成*）投稿内容をもとにOpenAIのAPIを利用した教義の生成<br><br>*旧verにある機能
## 旧verとの違い
- ajaxによる操作性の向上
- Dockerによる環境構築
## 技術スタック
- Django(Python)
- PostgreSQL
- jQuery
- Docker
- AWS
